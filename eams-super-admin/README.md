# EAMS 超级管理员后台

EAMS 系统的超级管理员后台，用于系统级管理和运营。

## 功能模块

### 一、数据面板
- 授权状态统计
- 激活量/新增量汇总
- 收入趋势图表
- AI算力使用统计
- 实时动态

### 二、系统设置
- 系统信息查看
- 参数设置
- 版本说明

### 三、服务商管理
- 服务商列表
- 服务商设置
- 等级管理
- 账户充值

### 四、级别管理
- 服务商等级
- 等级设置
- 折扣配置

### 五、开发管理
- 平台列表
- 平台设置
- API配置

### 六、财务管理
- 收支明细
- 充值明细
- 收入统计

### 七、AI算力中控
- 大模型管理
- 价格设置
- 使用统计

### 八、授权码管理
- 授权列表
- 批量生成
- 授权设置

### 九、安全管理
- 防破解设置
- 加密管理
- Ecode设置

## 启动命令

```bash
npm install
npm run dev
```

访问地址：http://localhost:5175

## 技术栈

- Vue 3
- TypeScript
- Element Plus
- Pinia
- Vue Router
- ECharts
