## 🎨 页面功能详情

### 1. Dashboard 数据仪表盘 (`/dashboard`)

**统计卡片** (4个):
- 总会话数 - 显示今日总对话数
- 已解决 - AI/人工已处理的会话
- 在线客服 - 当前在线客服人数
- 平均响应 - 平均响应时间

**图表区域**:
- 24小时对话趋势图 (折线图)
- 意图分布图 (饼图)
- 平台分布图 (饼图) - 展示各电商平台会话占比
- 满意度评分 - 星级评分 + 好评/中评/差评进度条

**实时会话动态**:
- 时间线展示最新会话
- LIVE 实时标识
- 刷新按钮

---

### 2. UnifiedInbox 统一收件箱 (`/inbox`)

**三栏布局设计**:

**左侧栏 - 店铺筛选 + 会话列表**:
- 平台筛选标签 (全部/淘宝/京东/拼多多/...)
- 店铺选择器 (按平台筛选)

**中间栏 - 聊天窗口**:
- 客户信息头部

- 快捷回复标签
- 输入框 + 发送按钮
- 转人工按钮

**右侧栏 - 用户资料**:
- 客户基本信息
- 订单历史
- 会话记录

## 🔧 技术栈

- **框架**: Vue 3 + TypeScript
- **UI库**: Element Plus
- **构建工具**: Vite
- **状态管理**: Pinia
- **图表**: ECharts
- **路由**: Vue Router

## 📱 响应式设计

- **桌面端**: 完整三栏布局

## 🎯 核心特性
