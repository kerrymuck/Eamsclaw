```
├── __init__.py          # 模块初始化，自动加载适配器
├── base.py              # 适配器基类 + 统一数据模型
├── registry.py          # 适配器注册表 + 元类
├── taobao.py            # 淘宝/天猫适配器
├── jd.py                # 京东适配器
└── amazon.py            # Amazon适配器
```

### 扩展新平台步骤
1. 创建适配器文件 `app/services/platform/{platform_id}.py`
2. 继承 `PlatformAdapter` 基类
3. 实现所有抽象方法
4. **自动注册到系统**（通过元类）
5. 配置平台参数到数据库
6. 前端添加平台图标和配置
