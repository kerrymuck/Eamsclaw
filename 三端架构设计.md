# EAMS 三端架构设计文档

## 系统架构概览

```
┌─────────────────┐  ┌─────────────────┐  ┌─────────────────┐
│   eams-client   │  │ eams-merchant   │  │ eams-frontend   │
│   买家客户端     │  │   商家管理端     │  │   平台管理端     │
│  (H5/小程序)     │  │  (PC/移动端)     │  │    (PC端)       │
└────────┬────────┘  └────────┬────────┘  └────────┬────────┘
         │                    │                    │
         └────────────────────┼────────────────────┘
                              │
              ┌───────────────▼───────────────┐
              │      eams-backend            │
              │    (FastAPI + PostgreSQL)    │
              │      WebSocket 服务          │
              └───────────────────────────────┘
```

## 三端定位

### 1. eams-client 买家客户端
- **目标用户**: 电商平台买家
- **使用场景**: 商品咨询、订单查询、售后服务
- **接入方式**: 嵌入电商平台（淘宝、京东、拼多多等）
- **技术方案**: H5响应式，支持iframe嵌入

### 2. eams-merchant 商家端
- **目标用户**: 电商商家、店铺运营人员
- **使用场景**: 店铺管理、客服配置、数据查看、消息回复
- **使用设备**: PC为主，支持平板和手机
- **技术方案**: Vue3响应式布局

### 3. eams-frontend 平台管理端（已有）
- **目标用户**: 平台管理员
- **使用场景**: 系统配置、用户管理、全局监控
- **使用设备**: PC端
- **升级需求**: 添加基础响应式支持
